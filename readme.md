## Technologies Used

- [Astro](https://astro.build/)

<br/>

## Installation

- Clone the repository <br/>

  ```
    git clone git@github.com:innerpage/website__dev.git
    cd website__dev
  ```

- Install dependencies <br/>
  ```
    ./init.sh
  ```

<br/>

## Build and Deployment

- Build the code <br/>

  ```
    ./build.sh
  ```

- Deploy the `prod` folder in your web server
