## Installation

- Clone the repository
  `git clone git@github.com:innerpage/website__dev.git cd website__dev`

- Install dependencies
  `./init.sh`

## Build and Deployment

- Build the code
  `./build.sh`

- Deploy the `prod` folder in your web server
